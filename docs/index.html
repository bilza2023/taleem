Taleem Documentation

This documentation explains the philosophy, structure, and usage of the Taleem slide system. It is intentionally short, direct, and focused on real usage rather than exhaustive reference material.

Taleem is not a generic presentation library. It is a teaching-first system designed to convert structured content into calm, readable educational slides.
